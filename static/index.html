avatar.now.sh

Get user avatar across social services.

From username

<img src="https://avatar.now.sh/:username" />

i.e https://avatar.now.sh/kikobeats

From email

<img src="https://avatar.now.sh/:email" />

i.e https://avatar.now.sh/josefrancisco.verdu@gmail.com

as JSON

Alternatively you can get the avatar url adding /json

// https://avatar.now.sh/kikobeats/json

{
  "url": "myrul"
}

Specifying the service

Each service has his own endpoint, so you can query it directly.

Twitter

<img src="https://avatar.now.sh/twitter/:username" />

i.e https://avatar.now.sh/twitter/kikobeats

GitHub

<img src="https://avatar.now.sh/github/:username" />

i.e https://avatar.now.sh/github/kikobeats

Gravatar

<img src="https://avatar.now.sh/gravatar/:email" />

i.e https://avatar.now.sh/email/josefrancisco.verdu@gmail.com
